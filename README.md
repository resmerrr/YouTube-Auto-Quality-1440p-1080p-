# YouTube Auto Quality (1440p / 1080p)

A userscript that automatically sets YouTube video playback quality 
to 1440p when available, falling back to 1080p.

## Installation

1. Install [Violentmonkey ](https://violentmonkey.github.io/) 
   ()
2. Click the link below to install the script:

   [⬇️ Install Script](../../raw/main/youtube-auto-quality.user.js)

3. Click **Confirm Installation** in the popup

## Features

- Prefers 1440p, falls back to 1080p
- Handles YouTube SPA navigation
- Retries automatically if player isn't ready
- Verifies the quality change applied
- Skips YouTube Shorts
- Lightweight and open source
